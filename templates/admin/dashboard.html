Platform Guide

Welcome to the College Event Management System (EMS). Use this dashboard to govern registrations and monitor the campus event catalog.

  • Deleting Events: Deleting an event will instantly and recursively purge all registrations associated with that event. Use this action with caution.
  • Capacity Constraints: The system automatically caps registration once the specified event seat capacity is reached. Students will see a "Slots Full" status on their dashboard.
  • Excel-Ready Views: The Registrations page lists student information, team names, and categories. Perfect for copy-pasting into spreadsheet tools.
